# SDL3 Learning Application

A C++23 application built with SDL3 for learning game development fundamentals. This project uses modern C++ practices, CMake for build management, and Conan for dependency management.

## Project Overview

This is a modular C++23 application designed to learn SDL3 graphics programming. The project features:
- Static linking of SDL3 (no runtime DLL dependencies)
- Separation of game logic into a reusable static library
- Comprehensive unit testing with Catch2
- CMake-based build system
- Conan package management
